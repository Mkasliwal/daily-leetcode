## Multidimensional Matrix in `javascript` 
 
JavaScript natively doesn't support multidimensional arrays. Despite the fact, `js` is powerful enough to facilitate operations and computations on multidimensional arrays without creating any overhead.
 
Multidimensional arrays in `js` can be implemented using array literal, that means `Array of arrays` _e.g.,_
```javascript
const multiDimensionalArray = [
    ["key1", "value1"],
    ["key2", "value2"],
    ["key3", "value3"],
    ["key4", "value4"],
];
 
console.log(multiDimensionalArray[1][1])
// output -> value2
```
